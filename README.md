# Real-Estate-Management-API
Real Estate Management API, emlak ilanlarını yönetmek, aramak, filtrelemek ve müşteriler ile emlakçılar arasında iletişim kurmak için geliştirilmiş bir RESTful API'dir.  

Proje Amacı:

Emlak ilanlarını sistematik şekilde yönetmek
Gelişmiş arama ve filtreleme özellikleri sunmak
Emlakçı ve müşteri yönetimi yapmak
İletişim ve sorgulama süreçlerini yönetmek
Kullanıcı Tipleri:

Admin (Yönetici): Tüm sistem üzerinde tam yetki
Agent (Emlakçı): Kendi ilanlarını yönetebilir, müşteri sorgularını görebilir
User (Müşteri): İlanları görüntüleyebilir, arama yapabilir, sorgu gönderebilir
